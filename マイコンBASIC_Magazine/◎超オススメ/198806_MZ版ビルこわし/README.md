【キー操作】  
移動：  
　　８  
　　↑  
４←┼→６  
　　↓  
　　２  
  
GRPH SFT_LOCKの機能は以下の通り。  
(1)PLAYモード GRPH      SFT  
   PLAY時　　 AGAIN     NEXT  
   GAMEOVER時 AGAIN     タイトルへ  
  
(2)EDITモード GRPH       SFT  
   画面作成時 START      CHANGE  
   PLAY時     AGAIN      －  
　 GAMEOVER時 画面作成へ タイトルへ  
  
【ゲーム内容】  
８，２，４，６で上下左右にブルドーザーを操作してダイナマイトを並べ替える  
ダイナマイトは一つ爆発すると縦横に連鎖して次々と爆発するので  
これを利用して1回の爆発ですべてのビルを壊す。
