# CMU-800を演奏しながらGRAM操作<br><br>
**使用方法**<br>
・EmuZ-2000 TF-Editionで状態ファイルを読み込み。<br>
・CMU-800プラグインのテンポを170とか好きなテンポに変更。<br>
・EmuZ-2000のシステムクロックを400MHzに変更。<br>
・Pを押せば再生開始。<br>
