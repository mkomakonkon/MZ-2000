# Enuz-2200のフォントを入れ替えてみた  
ATARI_美咲.ROMをFONT.ROMにリネームしてmz2200.exeと同じフォルダに置くとフォントが変わります。  
元のフォントのバックアップを忘れずに。  
ASCII.zipは作ったフォント確認用のテープイメージです。
