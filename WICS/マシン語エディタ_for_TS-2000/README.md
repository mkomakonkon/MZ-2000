# マシン語エディタ for TS-2000

I/O 1983年2月号に掲載のMZ-2000/80B用マシン語エディタをBASEに移植。<BR>
BASEの勉強が目的だったので中途半端だけど一旦終了予定。<BR>
 
### Ver.0.9
・ファンクションキーは動作しません。<BR>
・A～Fのキーアサインは以下の通り。<BR>
| 16進 | アサイン |
| --- | --- |
| A | - |
| B | + |
| C | . |
| D | Enter |
| E | E(通常キー) |
| F | F(通常キー) |
  
EとFのオリジナルの位置は押したくないキーが多すぎるので通常キーに変更
